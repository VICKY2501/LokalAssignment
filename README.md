# LokalAssignment
# Android Product Showcase App

This Android app showcases a list of products retrieved from the given API endpoint. Users can view a list of products and click on any product to see its details.

## Features

- Display a list of products with relevant information.
- View detailed information about a selected product.
- Graceful handling and communication of failures and retries.
- Utilizes Retrofit for API communication.
- Uses Material UI for a modern and responsive user interface.
- Integrates Picasso for image loading.

## Libraries and SDKs Used

- [Retrofit](https://square.github.io/retrofit/) - A type-safe HTTP client for Android and Java.
- [Material Components for Android](https://material.io/develop/android/docs/getting-started/) - UI components following the Material Design guidelines.
- [Picasso](https://square.github.io/picasso/) - A powerful image downloading and caching library for Android.

Usage
Launch the app.
Explore the list of products.
Click on a product to view its details.
Experience graceful handling of failures and retries.
